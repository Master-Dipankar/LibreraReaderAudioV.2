# 7.10

*根据读者的要求和反馈进行的重大更新
*新的语音合成(TTS)界面
*增强了对EPUB 3.0的支持，可以播放音频文件
*各种格式的多字搜索查询，包括PDF
*支持带槽智能手机
*状态栏图例现在具有三个选项可供选择：完整，紧凑和面向章节
*欢迎更多反馈和要求！
